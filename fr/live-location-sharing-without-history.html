Partage en direct privé

Partager la position en direct sans historique serveur

Parfois quelqu'un doit savoir où vous êtes maintenant : road trip familial, marche solo, rendez-vous, randonnée de groupe ou retour à la maison. Le partage App Cloud de GPS Logger vise ce moment sans transformer votre historique enregistré en trace cloud.

Réponse rapide

Le partage en direct GPS Logger envoie votre position actuelle aux suiveurs autorisés tant que le partage est actif. Il ne stocke pas d'historique serveur. La position actuelle est écrasée pendant le déplacement, supprimée à l'arrêt et protégée par un code réinitialisable et une limite de suiveurs.

Pourquoi ce n'est pas un historique

Une trace enregistrée est votre journal privé d'un parcours. Le partage en direct a un autre rôle : montrer où vous êtes maintenant. GPS Logger sépare ces deux idées.

Vos traces enregistrées restent sur l'appareil par défaut. App Cloud relaie seulement la position actuelle pendant que vous choisissez de diffuser.

Ce que voient les suiveurs

Les personnes avec votre code peuvent voir votre position actuelle sur une carte tant que le partage est actif. Leur app peut reconstruire une courte trace locale à partir des mises à jour successives, mais App Cloud stocke la position actuelle, pas toutes les positions historiques.

Ce que vous contrôlez

  • Démarrer et arrêter. Le partage commence seulement quand vous l'activez et s'arrête quand vous l'arrêtez.
  • Code. Le code est réinitialisable. La réinitialisation supprime l'ancien enregistrement et révoque les anciens suiveurs.
  • Limite de suiveurs. Choisissez combien de personnes peuvent suivre activement.
  • Destination. Utilisez App Cloud pour le partage géré ; l'app affiche les options disponibles au bon moment.

À l'arrêt du partage

Arrêter un partage App Cloud supprime la position en direct et les enregistrements temporaires des suiveurs. Si un appareil se déconnecte mal, les positions anciennes sont nettoyées automatiquement. Le lien de propriété du code reste pour éviter qu'une autre installation reprenne votre code actuel.

Quand c'est utile

  • Permettre à la famille de suivre une arrivée en road trip
  • Partager la progression pendant une marche, randonnée ou sortie solo
  • Coordonner un rendez-vous sans messages constants
  • Voir où se trouve la personne qui mène un groupe
