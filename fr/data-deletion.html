Suppression des données pour GPS Logger

Dernière mise à jour : 12 juin 2026

GPS Logger stocke vos parcours enregistrés sur votre appareil par défaut. NorthforceApps ne reçoit ni ne stocke votre historique de parcours privé. Cette page explique comment supprimer chaque type de données traitées par l'application.

Contact : northforceapps [at] protonmail [dot] com

Données locales (votre appareil)

Votre historique de parcours, vos points de parcours et vos paramètres d'application se trouvent dans la base de données de l'application sur votre appareil. Pour les supprimer :

  • Supprimez les parcours individuels dans l'application.
  • Effacez le stockage de l'application dans Paramètres Android → Applications → GPS Logger → Stockage.
  • Désinstallez l'application pour supprimer toutes les données stockées localement.

Partage en direct (App Cloud)

Lorsque le partage en direct est actif, seule votre position actuelle est stockée dans l'App Cloud et écrasée sur place. Il n'y a pas de piste de localisation côté serveur.

  • Pour supprimer la position en direct actuelle de l'App Cloud, ouvrez GPS Logger et arrêtez le partage en direct.
  • Pour révoquer un code de partage et supprimer cet enregistrement de partage cloud, ouvrez les Paramètres et réinitialisez ou renouvelez le code de partage. Cela supprime l'ancien nœud de suivi et révoque les abonnés existants.
  • Les positions en direct obsolètes sont automatiquement supprimées dans un délai de 7 jours, et les créneaux de spectateurs obsolètes dans un délai de 10 minutes, par une tâche de nettoyage planifiée.

Partage auto-hébergé (votre propre serveur)

Si vous configurez une URL de serveur HTTPS auto-hébergé, votre position actuelle est envoyée uniquement à ce serveur. NorthforceApps ne la reçoit jamais. Supprimez ces données de votre propre serveur et désactivez ou effacez l'URL de téléversement dans les Paramètres pour arrêter tout nouvel envoi.

Analyses et diagnostics de plantage

  • Vous pouvez désactiver les analyses anonymes et les rapports de plantage dans Paramètres → À propos.
  • La désactivation arrête toute collecte future. Les données de diagnostic déjà traitées sous forme agrégée peuvent ne pas vous identifier personnellement.
  • Pour toute question sur la suppression des diagnostics, contactez-nous à l'adresse de support ci-dessus.

Demande de suppression

La plupart des données sont locales uniquement ou supprimées automatiquement en arrêtant un partage, en réinitialisant le code de partage ou par le nettoyage automatique des données obsolètes. Pour toute demande de suppression restante que les contrôles intégrés à l'application ne peuvent pas traiter, envoyez un e-mail à NorthforceApps à l'adresse de support ci-dessus et nous y donnerons suite lorsque cela est techniquement possible.
