Privates Live-Sharing

Live-Standort teilen ohne serverseitigen Routenverlauf

Manchmal soll jemand wissen, wo du gerade bist: Roadtrip, Solo-Spaziergang, Abholung, Gruppenwanderung oder Heimweg. GPS Logger App Cloud ist für diesen Moment gebaut, ohne deinen gespeicherten Routenverlauf in eine Cloud-Spur zu verwandeln.

Kurzantwort

GPS Logger Live-Sharing sendet deine aktuelle Position an zugelassene Follower, solange das Teilen aktiv ist. Es speichert keinen serverseitigen Routenverlauf. Die aktuelle Position wird überschrieben, beim Stoppen entfernt und durch zurücksetzbaren Code plus Follower-Limit geschützt.

Was Follower sehen

Follower mit deinem Code sehen deine aktuelle Position auf einer Karte, solange du teilst. Ihre App kann aus aufeinanderfolgenden Updates eine kurze lokale Spur für die Anzeige bilden, aber App Cloud speichert die aktuelle Position, nicht den Verlauf aller Positionen.

Was du kontrollierst

  • Start und Stopp. Live-Sharing läuft nur, wenn du es einschaltest.
  • Freigabecode. Der Code ist zurücksetzbar; alte Follower verlieren dann Zugriff.
  • Follower-Limit. Du wählst, wie viele aktive Follower teilnehmen können.
  • Ziel. App Cloud für verwaltetes Teilen; optional kann eine eigene HTTPS-Server-URL genutzt werden, wenn die App diese Option freischaltet.

Aktuelle Position teilen, nicht den ganzen Verlauf

Nutze einen zurücksetzbaren Code, wähle ein Follower-Limit und stoppe das Teilen, wenn der Moment vorbei ist.
